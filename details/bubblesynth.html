Bubble Synth: Chance Composition

Jul 2024

Used: Tone.js, React, Javascript

Source Code

Bubbel Synth is a interactive music generator built with Javascript and Tone.js. It features a unique graphical interface with bubbles bouncing around to control and modulate the sound. The "physical" location of bubbles, at the time of sampling, determines how synth engine is triggered. This project is inspired by Brian Eno, chance composition, and experimental music.

The Demo

Key Features

Interactive Control: Select low/mid/high notes and instrument, drop the bubble anywhere on the canvas. The bubble will get a random acceleration and exchange velocity upon contact with the border or another bubble.

Customize Tonality: Select the root key and scale/chord type, the canvas will be divided to map bubble position to each note in that scale/chord, from left to right.

Effects: Select a effect bubble and drop it on canvas to modulate all sound. The parameters of the effect (e.g. cutoff of the filter, feedback of the delay) will modulate based on its position similar to note bubbles.
